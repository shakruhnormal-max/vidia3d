# 3D Video Overlay Editor

Браузерный редактор для наложения вращающихся псевдо-3D объектов на видео.

## Запуск

Откройте `index.html` в браузере **через локальный HTTP-сервер** (не напрямую как file://, так как Three.js требует CORS):

```bash
# Python 3
python3 -m http.server 8080

# Node.js (npx)
npx serve .

# VS Code: расширение Live Server
```

Затем откройте: http://localhost:8080

## Требования к браузеру

- Chrome 90+ / Edge 90+ / Firefox 90+ (рекомендуется Chrome)
- Поддержка WebGL 2.0
- Интернет-соединение (для загрузки Three.js с CDN при первом открытии)

## Функциональность

### Видео
- Форматы: MP4, WebM, MOV
- Drag & drop или кнопка «Видео»
- Воспроизведение / пауза (Space), перемотка

### Изображения → 3D объекты
- Форматы: PNG, JPG, JPEG, WEBP
- **Режим A**: прямоугольная экструдированная пластина с настраиваемой толщиной
- **Режим B**: форма по альфа-контуру изображения

### Удаление фона
- 4 метода: цветовой ключ, светлый/тёмный фон, контур по альфа
- Интерактивный превью с настройкой порога
- Клик на цвет для выбора ключевого цвета

### 3D-анимация
- Автовращение по осям X / Y / Z
- Скорость: 1–720°/сек
- Направление: по часовой / против
- Начальный поворот по всем осям

### Таймлайн
- Визуальный редактор дорожек
- Drag ручек для изменения startTime / endTime
- Перемещение всего бара (move drag)
- Масштабирование (кнопки + / −)
- Горизонтальный скролл (колёсико)

### Fade In / Fade Out
- Независимые параметры для каждого объекта
- Отображаются на таймлайне треугольными зонами

### Управление слоями
- До 20+ объектов
- Видимость, блокировка, удаление
- Миниатюры изображений

### Экспорт
- Формат: WebM (VP9) — браузерный стандарт без серверных зависимостей
- Разрешение: 720p / 1080p / 1440p / 4K
- FPS: 24 / 30 / 60
- Покадровый рендеринг через Three.js + MediaRecorder

> **Примечание об экспорте**: Браузерный MediaRecorder создаёт файл WebM (VP9).
> Для конвертации в MP4 (H.264) используйте HandBrake, FFmpeg или любой онлайн-конвертер.
> WebM поддерживается всеми современными браузерами и видеоплеерами.

### Горячие клавиши
| Клавиша | Действие |
|---------|----------|
| Space | Воспроизведение / пауза |
| Delete / Backspace | Удалить выбранный объект |
| ← → ↑ ↓ | Переместить объект (1px) |
| Shift + ← → ↑ ↓ | Переместить объект (10px) |

### Сохранение проекта
- Сохранить: экспортирует JSON с параметрами всех объектов
- Открыть: восстанавливает проект (изображения хранятся как base64 dataURL)

## Стек
- HTML5 / CSS3 / Vanilla JavaScript
- Three.js r128 (3D рендеринг)
- Canvas API (удаление фона, таймлайн)
- MediaRecorder API (экспорт)
